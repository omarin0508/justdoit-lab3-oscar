# Programación Web II

## Descripcion del curso

Este repositorio contiene la estructura documental del curso **Programación Web II**. Su propósito es organizar de forma clara y académica los laboratorios, evidencias, notas de trabajo y documentos finales que se desarrollen durante el curso.

La documentacion se mantendra en formato Markdown para facilitar la escritura, revision y posterior exportacion a PDF cuando sea necesario entregar evidencias formales.

## Metodologia de trabajo

La metodologia de trabajo se basa en documentar cada laboratorio de manera ordenada, separando el desarrollo tecnico de las evidencias academicas. Cada laboratorio contendra su propio espacio de trabajo documental, incluyendo objetivos, avances, capturas de pantalla, notas y archivos PDF generados para entrega.

Esta organizacion permite mantener un historial claro del proceso de aprendizaje, registrar los avances realizados y conservar evidencias verificables del trabajo desarrollado.

## Uso de IntelliJ IDEA

Para el desarrollo de codigo del curso se utilizara **IntelliJ IDEA** como entorno principal de trabajo. Este IDE permitira administrar proyectos, revisar estructura de archivos, ejecutar aplicaciones y trabajar con herramientas relacionadas con el desarrollo web en Java y Spring Boot cuando corresponda.

En esta etapa inicial no se modifica codigo fuente ni configuracion de Spring Boot. La estructura creada corresponde unicamente a la documentacion academica del curso.

## Uso de Markdown

Markdown sera el formato base para documentar los laboratorios y evidencias. Su uso permite redactar documentos claros, portables y faciles de convertir a PDF.

Cada laboratorio incluira archivos `.md` para describir objetivos, procedimientos, avances y evidencias. Las imagenes y capturas de pantalla se almacenaran en carpetas dedicadas para mantener una estructura ordenada.
